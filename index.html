ex 1

Escreva um programa que solicita ao usuário um número inteiro positivo N e calcula e imprime a soma dos N primeiros números pares e a soma dos N primeiros números ímpares separadamente. Por exemplo, se o usuário inserir 5, o programa deve imprimir a soma dos primeiros 5 números pares e a soma dos primeiros 5 números ímpares.

ex 2

Escreva um programa que solicita ao usuário um número inteiro positivo N e imprime os primeiros N números da sequência de Fibonacci. A sequência de Fibonacci é uma série de números em que cada número é a soma dos dois anteriores, começando por 0 e 1. Por exemplo, os primeiros 8 números da sequência de Fibonacci são: 0, 1, 1, 2, 3, 5, 8, 13.

ex 3

Escreva um programa que solicita ao usuário um número inteiro positivo N e verifica se N é um número primo ou não. Um número primo é aquele que é divisível apenas por 1 e por ele mesmo. O programa deve imprimir uma mensagem indicando se o número é primo ou não. Por exemplo, se o usuário inserir 7, o programa deve imprimir que 7 é um número primo, enquanto se o usuário inserir 6, o programa deve imprimir que 6 não é um número primo.
